# Коллекция скриптов для Scriptable

Этот репозиторий содержит набор мощных и красивых JavaScript-виджетов для приложения Scriptable на iPhone и iPad.

## Что такое Scriptable?
Scriptable — это бесплатное приложение для iOS/iPadOS, которое позволяет писать и запускать JavaScript-код прямо на устройстве. С его помощью можно создавать кастомные виджеты на домашний экран и экран блокировки, автоматизировать задачи, получать данные из интернета и делать уведомления — всё без компьютером не требуется.

## Установка и использование
1. Установите приложение Scriptable из App Store
2. Скопируйте код нужного скрипта из этого репозитория в новый скрипт в Scriptable
3. При необходимости отредактируйте настройки внутри скрипта (API-ключи, адреса серверов и т.д.)
4. Добавьте виджет на экран через стандартное меню виджетов iOS
5. Готово!

## Скрипты в репозитории

### 1. Steam Wishlist (Виш-лист Steam со скидками)
**Файл:** `Steam Wishlist.scriptable`

Красивый виджет, который показывает игры из вашего вишлиста Steam с актуальными ценами и скидками.  
Поддерживает все размеры виджетов: маленький, средний, большой (домашний экран) и прямоугольный/строковый (экран блокировки).

Особенности:
- Сортировка по размеру скидки — самые выгодные игры сверху
- Показ обложек игр в средних и больших виджетах
- Кэширование данных на 12 часов
- Работает на русском языке (цены в рублях с символом)

Настройка:
- Вставьте свой Steam API Key и SteamID64 (можно передать как параметр виджета)
- Готово — скидки сами найдутся

### 2. Crypto Price Pro (Криптовалютный виджет + портфель)
**Файл:** `Crypto Price.scriptable`

Полноценный крипто-виджет с двумя колонками, портфелем и уведомлениями.  
Один из самых продвинутых открытых крипто-виджетов для Scriptable.

Возможности:
- Отслеживание любых монет через CoinGecko (BTC, ETH, TON, SOL, NEAR, ADA и любые другие)
- Управление портфелем: добавление/удаление монет, расчёт общей стоимости
- Две колонки: основные монеты слева + либо дополнительные монеты, либо топ-позиции портфеля справа
- Уведомления при изменении цены монеты или стоимости всего портфеля на заданный %
- Полноценное меню настроек прямо в приложении (добавить монету, включить/выключить уведомления и т.д.)
- Кэширование, автовосстановление после ошибок, красивые иконки

Ничего настраивать не нужно — работает сразу после установки.

### 3. Server Status (Статус серверов)
**Файл:** `Server status.scriptable`

Простой и надёжный виджет для мониторинга доступности ваших серверов, сайтов или домашних устройств.

Что делает:
- Проверяет несколько URL (можно добавить свои)
- Показывает статус: онлайн, офлайн, неизвестно
- Обновляется каждые 5 минут (можно изменить)
- Показывает время последнего обновления

Идеально для:
- Домашнего NAS / медиасервера
- Minecraft-сервера
- Личного сайта или VPN
- Любого сервиса, который отвечает на HTTP-запрос

Настройка:
Просто отредактируйте массив servers в начале скрипта — укажите свои адреса и названия.
